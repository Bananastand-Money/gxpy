GXVOXE class¶

class GXVOXE(wrapper=None)¶

GXVOXE class.

GXVOX evaluator class. Used to sample values from the voxel.

classmethod create(vox)¶

Create a handle to an GXVOXE object

Parameters:vox (GXVOX) – GXVOX Object
Returns:GXVOXE handle, terminates if creation fails
Return type:GXVOXE

New in version 6.3.

Check if the instance of GXVOXE is null (undefined)`

Returns:True if this is a null (undefined) instance of GXVOXE, False otherwise.
classmethod null()¶

A null (undefined) instance of GXVOXE

Returns:A null GXVOXE
profile(vv_x, vv_y, vv_z, vv_d, interp)¶

Extract a profile of data along points provided.

Parameters:
  • vv_x (GXVV) – X GXVV (must be double)
  • vv_y (GXVV) – Y GXVV (must be double)
  • vv_z (GXVV) – Z GXVV (must be double)
  • vv_d (GXVV) – D GXVV (must be double)
  • interp (int) – VOXE_EVAL

New in version 6.3.

value(x, y, z, interp)¶

Get a value at a specific point

Parameters:

  • y (float) – Y Location
  • z (float) – Z Location
  • interp (int) – VOXE_EVAL
Returns:

Value at the point or DUMMY if not valid

float

Extract a profile of data along a vector

Parameters:

  • oy (float) – Y Origin
  • oz (float) – Z Origin
  • vx (float) – X Delta
  • vy (float) – Y Delta
  • vz (float) – Z Delta
  • vv (GXVV) – Data GXVV (must be double)
