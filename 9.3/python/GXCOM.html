GXCOM class¶

class GXCOM(wrapper=None)¶

GXCOM class.

This class is used to communicate with external serial devices. It allows the setting of timeouts.

classmethod create(port, baud, data_size, parity, stop_bits, flow_control, time_out)¶

Create GXCOM object.

Parameters:
  • port (str) – Port name to open (“COM1” is example)
  • baud (int) – COM_BAUD
  • data_size (int) – COM_DATASIZE
  • parity (int) – COM_PARITY
  • stop_bits (int) – COM_STOPBITS
  • flow_control (int) – COM_FLOWCONTROL
  • time_out (int) – Timeout in Ms (500)
Returns:

GXCOM Object
Return type:

GXCOM
